# lankaqr

LankaQR encoder, decoder, and validator for TypeScript and JavaScript.

> **Status:** This is a name reservation. The real implementation ships in **v1.0.0** (planned weekend of 14–15 Jun 2026).
>
> See [the project page](https://github.com/lakshitha0526/lankaqr) for status.

## What this will be

A small, typed, zero-dependency library for encoding, decoding, and validating LankaQR payloads, conforming to:

- LANKAQR Specification V1.3 (Central Bank of Sri Lanka, January 2026)
- EMV® QR Code Specification for Payment Systems — Merchant-Presented Mode v1.1 (EMVCo, November 2020)

Part of the `lk-*` family of Sri Lankan developer tools.
